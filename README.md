## KalasinHospital-emr-exam
การสอบเขียน SQL และออกแบบ UI/UX Dashboard สำหรับห้องตรวจ<br>
**จัดทำโดย:** [นายธนดล ศรีปัญญา] | **รหัสนักศึกษา:** [671424005003-9]<br>

## 💻 ส่วนที่ 1: Database & SQL Query
ส่วนนี้คือชุดคำสั่ง SQL สำหรับสร้างฐานข้อมูลจำลอง (`HIS_TEST`) และคำสั่งดึงข้อมูลรายงานการวิจัยตามเงื่อนไขของโจทย์<br>

📌 วิธีการรันโค้ดเพื่อทดสอบ<br>
1. อาจารย์สามารถนำไฟล์หรือโค้ดใน ** HIS_TEST.sql ** ไป Import เข้าหรือรันใน phpMyAdmin เพื่อสร้างตารางและข้อมูลจำลองทั้งหมดในครั้งเดียว
2. จากนั้นนำโค้ดใน ** SQL_Query ** ไปรันเพื่อดูผลลัพธ์การ Query ข้อมูลครับ<br>
